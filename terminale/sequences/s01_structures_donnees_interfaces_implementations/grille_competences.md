# Grille de compétences - structures de données

## Compétences

- Distinguer interface et implémentation.
- Choisir pile ou file selon l'ordre de retrait.
- Utiliser un dictionnaire pour un accès par clé.
- Représenter un graphe par liste d'adjacence ou matrice.
- Identifier les cas limites.

## Niveaux

- Socle : la structure est reconnue.
- Standard : le choix est justifié.
- Expert : les limites et coûts sont discutés.
