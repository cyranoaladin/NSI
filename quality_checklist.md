# Checklist qualité pédagogique (pré-pub)

## Documenter
- [ ] Situation-problème en premier dans chaque document produit.
- [ ] Objectifs explicites.
- [ ] Alternance activité / formalisation / exemples / exercices.
- [ ] Erreurs fréquentes présentes.
- [ ] Auto-évaluation présente.
- [ ] Extension avancée présente.
- [ ] Aides progressives présentes.

## Métadonnées
- [ ] Source définie.
- [ ] Niveau défini.
- [ ] Statut défini.
- [ ] Thème et notion cohérents.
- [ ] Version renseignée.

## Technique
- [ ] Chaque séquence contient TP/TD/éval/corrigé/QCM/guide/ressource Python.
- [ ] Scripts Python exécutables et tests associés.
- [ ] Vérification de liens internes sans erreur.

## Gouvernance
- [ ] Données personnelles supprimées/anonymisées.
- [ ] Données obsolètes identifiées.
- [ ] Aucune ressource isolée hors séquence.
