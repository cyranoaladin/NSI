# Charte graphique et pédagogique NSI-enseignement

## Principes pédagogiques communs

- Une séquence = {cours élève, trace écrite, TD, TP, fiche méthode, aides, corrigés, guide professeur, évaluation, QCM, scripts/tests}.
- Tous les documents commencent par une **situation-problème**.
- Alternance systématique: activité → formalisation → exemples → exercices.
- Chaque document inclut :
  - section erreurs fréquentes
  - auto-évaluation
  - prolongement/extension (niveau avancé)
  - aides progressives (niveau fragiles)

## Charte visuelle

- Couleur primaire : `#102A43`
- Couleur secondaire : `#1A75BC`
- Accent : `#F4B400`
- Fond : dégradé subtil `linear-gradient(180deg, #f3f8ff 0%, #ffffff 45%, #eef2f9 100%)`
- Typographies :
  - Titres : `"Merriweather", "Georgia", serif`
  - Texte courant : `"Lato", "Segoe UI", sans-serif`
  - Code : `"Fira Code", "Consolas", monospace`

## Style par document (LaTeX / Markdown)

- Numéroter les séquences (S01, S02, ...).
- Ajouter un entête normalisé avec métadonnées (niveau, thème, notion, objectifs).
- Utiliser la voix : **élève-centrique**, actions concrètes, mini-projets.
- Encadrer : objectif(s), erreur(s) fréquentes, méthode, auto-évaluation.
