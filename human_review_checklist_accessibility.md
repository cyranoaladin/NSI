# Checklist accessibilité et différenciation

Cette checklist prépare une relecture humaine d'accessibilité. Elle ne rend aucune ressource publiable seule.

## Lisibilité

- Les titres structurent réellement le document.
- Les tableaux sont lisibles en impression.
- Les consignes longues sont segmentées.
- Les notations sont expliquées avant usage.

## Différenciation

- Une entrée socle est prévue.
- Un parcours standard est prévu.
- Une extension experte est prévue.
- Les aides progressives ne donnent pas directement la réponse.
- La version aménagée conserve les capacités visées.

## Accessibilité

- Les données essentielles ne reposent pas uniquement sur la couleur.
- Les exemples sont compréhensibles sans support externe non fourni.
- Les documents élèves ne citent pas de documents professeur comme ressources nécessaires.
- Les critères de réussite sont observables.
