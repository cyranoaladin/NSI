## P00 - Rentrée, diagnostic, outils, méthode NSI

- Mois : septembre.
- Volume horaire : 4 h.
- Projet : 1 h, carnet de bord et choix d'un thème personnel fictif.
- Capacités officielles : P-HIST-01, P-LANG-01.
- Documents à produire : fiche méthode environnement, diagnostic Python, fiche carnet de bord, guide professeur.
- TD : diagnostic logique, variables, boucles, lecture de code.
- TP : installation éditeur, tests simples, dépôt local non publié.
- Évaluation : diagnostic non classant.
- Micro-projet : mini-carnet de bord numérique avec données fictives.
- Différenciation : parcours socle pour prise en main, parcours expert avec fonctions.
- Ressources Drive candidates : dossiers de reprise NSI et TPs Python référencés dans `drive_inventory.csv`.
- Points de vigilance : ne pas confondre diagnostic et sélection ; anonymiser toute donnée élève.
- Remédiation : ateliers courts sur variables, conditions, boucles.
