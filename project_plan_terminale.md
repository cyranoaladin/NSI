# Plan projets NSI Terminale - Tunisie 2026-2027

- Statut : prototype annuel non publiable.
- Niveau : terminale
- Total horaire NSI estimé : 210 h
- Total projets : 60 h
- Ratio projet : 28.6 %
- Référence calendrier : `calendar_2026_2027_tunisia.md`.
- Décision : les projets préparent bac pratique, écrit et Grand Oral, sans rendre les ressources publiables.

## Principes

Les projets occupent plus d'un quart de l'horaire. Ils sont distribués sur l'année pour éviter une surcharge en fin de parcours et pour soutenir le Grand Oral. Les sujets doivent rester compatibles avec les capacités officielles, utiliser des données fictives et produire des preuves de tests.

## Projet intermédiaire

- Période : fin janvier, avant Ramadan.
- Durée cumulée : environ 24 h de projet déjà engagées.
- Sujet recommandé : application de gestion fictive avec structures abstraites, classes, graphes ou arbres, tests et analyse de coût.
- Livrables : code modulaire, tests, schéma des structures, fiche de cas limites, démonstration courte.
- Critères : interface claire, implémentation justifiée, tests de cas limites, trace d'exécution, absence de données personnelles.
- Différenciation : version minimale centrée sur pile/file/dictionnaire ; version standard avec arbre ou graphe ; version experte avec comparaison de représentations.

## Projet final

- Période : mars à juin, jalons légers pendant Ramadan, finalisation en T19.
- Sujet recommandé : étude d'un système numérique fictif combinant données, base relationnelle, graphe/réseau, sécurité et algorithme.
- Version minimale : modèle de données, opérations principales, tests, notice.
- Version standard : interface de fonctions, base SQLite, parcours ou algorithme, module de tests.
- Version experte : optimisation, analyse de complexité, robustesse, support Grand Oral argumenté.
- Livrables : dossier technique, code, tests, base fictive, grille RGPD, fiche bac pratique, support Grand Oral.

## Carnet de bord

Chaque élève documente objectifs, décisions, erreurs rencontrées, tests ajoutés, sources utilisées et contribution personnelle. Le carnet alimente la préparation Grand Oral et permet de vérifier que le projet n'est pas une juxtaposition de fragments non maîtrisés.

## Soutenance

- Durée indicative : 8 minutes de présentation et 5 minutes de questions en Terminale.
- Attendus : problématique, architecture, données fictives, structure ou algorithme central, démonstration, tests, limites, lien sociotechnique possible.
- Place de l'oral : micro-oraux après T03, T08, T13 ; soutenance intermédiaire en janvier ; soutenance finale en juin.

## Grille d'évaluation projet

| Critère | Indicateurs observables | Niveau fragile | Niveau attendu | Niveau avancé |
|---|---|---|---|---|
| Modélisation | structures, schéma, invariants | choix peu justifié | modèle lisible | comparaison argumentée |
| Code et tests | modularité, robustesse, cas limites | tests rares | tests utiles | tests de régression et erreurs |
| Algorithmique | correction, coût, terminaison | trace incomplète | justification correcte | complexité discutée |
| Données et sécurité | données fictives, RGPD, risques | risques peu vus | risques maîtrisés | analyse critique |
| Bac pratique | autonomie, fonctions, debug | besoin d'aide constant | résolution guidée autonome | transfert à sujet nouveau |
| Grand Oral | problématique, clarté, recul | exposé descriptif | argumentation claire | recul scientifique et sociétal |

## Gestion des groupes

- Groupes de 2 ou 3 élèves, avec responsabilités explicites.
- Rôles tournants : conception, tests, documentation, présentation.
- Les contributions individuelles sont notées dans le carnet de bord.
- Les données sont fictives ; toute donnée réelle est interdite dans les exports et supports.

## Différenciation

- Élèves fragiles : sujet final réduit, squelette de modules, tests fournis au départ, entretiens courts.
- Élèves au rythme standard : autonomie sur choix d'implémentation et revue de code.
- Élèves avancés : comparaison de représentations, optimisation, extension réseau ou sécurité, préparation orale plus analytique.
